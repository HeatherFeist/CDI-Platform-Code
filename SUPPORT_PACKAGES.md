## ğŸª™ Marketplace Storefront Integration

### Understanding the Unified Marketplace Model

The CDI Marketplace is **one unified store** where all merchants sell together, but each merchant has their own:
- **Merchant Coin** (unique to their business)
- **Storefront Profile** (their branded section)
- **Product Catalog** (their offerings)

**Key Concept**: Merchant coins serve as both **currency** AND **navigation/discovery tool** to find that seller's products.

### Database Schema for Coin-Storefront Linking

```sql
-- Merchants Table
CREATE TABLE merchants (
  id UUID PRIMARY KEY,
  user_id UUID REFERENCES users(id),
  business_name VARCHAR(255),
  slug VARCHAR(255) UNIQUE, -- URL-friendly name
  description TEXT,
  logo_url VARCHAR(500),
  banner_url VARCHAR(500),
  category_ids UUID[], -- Array of category IDs
  location_city VARCHAR(100),
  location_state VARCHAR(50),
  website_url VARCHAR(500),
  phone VARCHAR(20),
  email VARCHAR(255),
  support_tier VARCHAR(50), -- 'foundation', 'growth', 'enterprise'
  featured BOOLEAN DEFAULT FALSE,
  verified BOOLEAN DEFAULT FALSE,
  rating DECIMAL(3,2),
  total_reviews INTEGER DEFAULT 0,
  created_at TIMESTAMP DEFAULT NOW(),
  updated_at TIMESTAMP DEFAULT NOW()
);

-- Merchant Coins Table
CREATE TABLE merchant_coins (
  id UUID PRIMARY KEY,
  merchant_id UUID REFERENCES merchants(id) UNIQUE, -- One coin per merchant
  coin_name VARCHAR(100), -- "Fresh Bread Bakery Coin"
  coin_symbol VARCHAR(20) UNIQUE, -- "BAKERY_COIN"
  coin_icon_url VARCHAR(500),
  current_price DECIMAL(10,2) DEFAULT 1.00,
  total_supply DECIMAL(18,2),
  circulating_supply DECIMAL(18,2),
  market_cap DECIMAL(18,2),
  discount_percentage INTEGER DEFAULT 0, -- e.g., 10 for 10% off
  benefits TEXT[], -- Array of benefit descriptions
  created_at TIMESTAMP DEFAULT NOW(),
  updated_at TIMESTAMP DEFAULT NOW()
);

-- Products Table
CREATE TABLE products (
  id UUID PRIMARY KEY,
  merchant_id UUID REFERENCES merchants(id),
  name VARCHAR(255),
  description TEXT,
  price_usd DECIMAL(10,2),
  price_merchant_coin DECIMAL(10,2), -- Calculated based on coin value
  images_urls TEXT[],
  category_id UUID,
  inventory_count INTEGER,
  sku VARCHAR(100),
  created_at TIMESTAMP DEFAULT NOW(),
  updated_at TIMESTAMP DEFAULT NOW()
);

-- Storefront Views (Analytics)
CREATE TABLE storefront_views (
  id UUID PRIMARY KEY,
  merchant_id UUID REFERENCES merchants(id),
  viewer_id UUID REFERENCES users(id), -- NULL if not logged in
  source VARCHAR(100), -- 'coin_directory', 'product_page', 'search', 'wallet'
  coin_clicked BOOLEAN DEFAULT FALSE, -- Did they arrive via coin link?
  timestamp TIMESTAMP DEFAULT NOW()
);
```
