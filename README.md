# CDI Platform Code

This repository contains the source code for the entire Constructive Designs Inc. platform.

## Structure

- `apps/smart-hub`: The main CDI Smart Hub application.
- `apps/image-editor`: The CDI Image Editor application.
- `apps/cdi-monorepo`: The monorepo containing the Quantum Wallet and other shared packages.
- `apps/home-reno-pro`: The Home Reno Pro application.
- `apps/marketplace`: The Constructive Designs Marketplace application.

## Getting Started

Each application is located in its own directory under `apps/`. To work on a specific application, navigate to its directory and follow its specific instructions (usually `npm install` and `npm run dev`).
